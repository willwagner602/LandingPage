About Me

I'm a software engineering student interested in solving challenging problems. Professionally, I have worked on data automation for one of the country's largest education organizations. Below, you'll find examples of a few projects I've built in my free time.

CamPose

A short demo using OpenCV to identify and display camera relative position and pose. Take a look at the source code and run the demo on github.

Aircraft Spotter

A plane identification quiz, using open source images from wikimedia. Take it for a ride.

PyJobs

PyJobs uses a group of RESTful APIs to aggregate jobs from popular job search sites.

Thu2D

Thu2D is a javascript implementation of Terry Pratchett's Thud, a chess-like fantasy game. It's powered by a Python RESTful API.

Thud

Thud is the python backend supporting Thu2D. Documentation of the RESTful API can be found on its GitHub Page.
